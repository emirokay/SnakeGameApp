# SnakeGameApp
The Snake Game App is a SwiftUI implementation of the classic Snake game, featuring a clean interface with both light and dark mode support. Built using the Model-View-ViewModel (MVVM) architecture, it ensures code organization and maintainability, while leveraging SwiftUI and Combine for a smooth gameplay experience.

## Features
- **Classic Gameplay**: Navigate the snake to consume food, growing in length with each bite.
- **Score Count**: Tracks and displays the player's score during gameplay.
- **Customizable Settings**:
  - **Cell Size**: Choose from Small, Medium, or Large grid cells.
  - **Game Speed**: Select Slow, Medium, or Fast timer intervals.
- **Dynamic UI Updates**: Interface adjusts in real-time based on user settings, ensuring an engaging and seamless experience.
- **Game Over and Win Alerts**: Receive notifications upon game completion or achieving a high score.
- **Dark Mode Support**: Automatically adapts to the user's system settings for a consistent experience in both light and dark themes.

## Technologies Used
- **Language**: Swift
- **Frameworks**: SwiftUI, Combine
- **Architecture**: MVVM (Model-View-ViewModel)

## Project Structure
- **Models**: Defines the game's data structures and logic, including snake positioning and food placement.
- **ViewModels**: Manages game state and business logic, ensuring smooth interactions between the model and views.
- **Views**: Contains SwiftUI views for the user interface, including gameplay and settings screens.

## Getting Started

### Prerequisites
- **Xcode**: Version 12.0 or higher
- **iOS**: Deployment target of iOS 14.0 or higher
- **macOS**: Version 10.15 or higher

### Steps

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/emirokay/SnakeGameApp.git
2. Navigate to the Project Directory:
	```bash
	cd SnakeGameApp
3. Open in Xcode:
	```bash
	open SnakeGameApp.xcodeproj
4. Build and Run:
- Select a simulator or connect a physical device.
- Click the 'Run' button in Xcode to build and launch the app.
